# 8plus 諮詢平台

## 截圖
![8plus Web](/og/labs/8plus/web.png)
![8plus 內容](/og/labs/8plus/content.png)

**Tags**: vibe coding, Next.js, Google Calendar, Vercel, UI/UX Design

## 簡介
結合多年產品與開發經驗，打造一個能夠快速預約的諮詢平台，協助使用者就職涯與程式開發相關議題進行深度討論。透過資料分析背景與長期實務經驗，聚焦於挖掘每位使用者的能力特點，並給出具體可行的規劃路徑。

## 特色亮點
- 系統化的預約流程，與 Google Calendar 無縫整合
- 以使用者目標為中心的諮詢框架與問題引導
- 極簡清晰的介面，降低學習成本，提升完成率

## 技術棧
- 前端框架：Next.js
- 佈署平台：Vercel
- 外部整合：Google Calendar API
- 設計：UI/UX 設計與互動細節優化
