# 光復協作平台（e-cooperative）

**Tags**: vibe coding, Next.js, Vercel, Python Flask API, UI/UX Design

## 專案背景
2025/09/23 花蓮光復區發生馬太鞍溪堰塞湖災害。透過社群與志工串聯，啟動協作平台的設計與原型開發，目標為整合資訊、提升溝通效率、並協助資源媒合。

## 目標與價值
- 整合多來源災情資訊，建立標準化資料模型
- 提供志工與居民的互助與媒合機制
- 以輕量可維護為前提，快速上線與迭代

## 技術棧
- 前端：Next.js（Vercel 佈署）
- 後端：Python Flask API
- 設計：UI/UX 設計與資訊架構
