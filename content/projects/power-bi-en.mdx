# Power BI — Market & Disease Relationship Exploration

**Tags**: Python, Power BI

Based on public datasets, this Lab explores possible relationships between stock market movement, employment market changes, and specific disease patterns. Python handles data processing and feature engineering, while Power BI provides the interactive dashboard layer.

## Focus
- Data cleaning and feature engineering workflow
- Indicator and time-series visualization
- Filtering and interactive analysis
