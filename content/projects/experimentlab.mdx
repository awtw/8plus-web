# ExperimentLab — 醫學分析原型平台

## 截圖
![ExperimentLab Web](/og/labs/experimentlab/web.png)
![ExperimentLab 分析](/og/labs/experimentlab/analysis.png)

**Tags**: Vue 3, UI/UX Design, Analysis Algorithm

## 專案簡介
協助醫師在申請研究計畫時使用的原型平台，透過指標數據（例如覆膜透析程度、基因表現）建立初步預測模型，評估疾病惡化或好轉的風險趨勢。

## 功能重點
- 指標資料視覺化呈現與趨勢追蹤
- 基礎模型推估與可調參數
- 記錄與匯出研究用報表

## 技術棧
- 前端：Vue 3
- 設計：UI/UX 設計、資料視覺化
