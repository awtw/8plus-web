# SmartCommunityBackEnd — Smart Community Backend

![SmartCommunityBackEnd Web](/og/labs/smart-community-backend/web.png)

**Tags**: .NET, PostgreSQL, Redis, Message Queue, Docker

A backend service for smart community applications, covering residents, devices, community services, and incident reporting. The system emphasizes reliability and observability.

## Core Modules
- User / resident / permission management
- Device and sensor data collection
- Incident reporting and workflow: task assignment, processing, and reporting
- Notification center: email, SMS, and push notifications

## Stack
- Platform: .NET
- Data: PostgreSQL, Redis
- Queue: RabbitMQ / Kafka depending on deployment setup
- Deployment: Docker / docker-compose
