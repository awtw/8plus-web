# Code Review Checklist

## 架構與邊界

- [ ] 模組職責是否單一、依賴方向是否合理
- [ ] API 契約是否版本化、錯誤碼是否一致
- [ ] 設定與 secret 是否外置

## 可靠性

- [ ] 重試、timeout、冪等是否處理
- [ ] 日誌是否可追蹤 request id
- [ ] 邊界條件與空狀態

## 安全

- [ ] 輸入驗證與輸出編碼
- [ ] 授權檢查在 server 端
- [ ] 敏感資料不落 log
