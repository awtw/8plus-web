# Guangfu Collaboration Platform (e-cooperative)

**Tags**: vibe coding, Next.js, Vercel, Python Flask API, UI/UX Design

## Background
On 2025/09/23, the Matai'an Creek barrier lake disaster affected the Guangfu area in Hualien. This Lab prototypes a collaboration platform that connects social networks and volunteers to integrate information, improve communication efficiency, and support resource matching.

## Goals & Value
- Integrate disaster information from multiple sources into a standardized data model
- Support mutual aid and resource matching between volunteers and residents
- Launch and iterate quickly with a lightweight, maintainable architecture

## Stack
- Frontend: Next.js deployed on Vercel
- Backend: Python Flask API
- Design: UI/UX design and information architecture
