我的故事
自學驅動的工程之路：從生醫到全端與雲端

我是一個從小就熱愛自學的人。雖然因為家庭期待曾在許多領域嘗試，但最終仍不斷探索，找到自己真正的熱情所在。小時候常參加藝術比賽，培養了對美感與設計的直覺與執著；大學主修生物與化學，雖然成績不錯，卻意識到這並不是自己想長期投入的職涯方向。

進入交大與技術轉向
後來進入交大分子醫學與生物工程所，在院長的資工實驗室專注於基因定序與生物分析。這段時間除了使用 R 與 Python 進行數據分析，還選修了許多資工課程。在一次課堂專案中，我完成了一個前後端整合的需求，受到教授肯定並開始承接實際案子。從此，我踏上了自學程式的道路，從基礎的 HTML、CSS、JavaScript、PHP，到後來的 Angular、React、Vue，甚至 Express.js 後端開發，都一一嘗試並逐漸精進。

替代役與產品實作
在替代役期間，我每天利用下班後的時間補強資工知識，完成多個作品集，甚至建立了自己的品牌「1914」，並嘗試整合 LINE、Facebook Chatbot、Google Dialogflow API，開發語意辨識客服機器人並部署至 Heroku。

職涯挑戰與成長
退伍後，我選擇挑戰純後端工程師職位，不僅高分通過考核，也獲得公司給予的技術探索與發揮空間。這段經歷讓我更清楚自己的不足與方向：持續學習、主導新技術導入、建立問題解決的思維模式。下班之餘，我仍透過線上課程不斷充實自己，保持成長。

電商與 SaaS 的實戰體悟
進入電商產業後，我更深入理解了 B2C 的商業需求與產品挑戰，也在實務中落地多項關鍵能力：

SLA 與 7×24 小時服務保障、故障切換與韌性設計
藍綠部署與漸進式發布，降低上線風險
SaaS 架構設計、租戶隔離與版本管理
雲端資源整合與成本優化（觀測、配額、預算控管）
跨國部署與法規合規（資料駐留、稅務與隱私）
Git Flow、Feature Toggle、Grafana 即時監控等工程實踐
我也曾參與前端大改版，推動共用元件化與 npm library，建立跨部門可用的開發模式。這些經歷讓我更全面地理解 SaaS 服務與雲端架構的威力。

自由接案與架構觀
除了正職經驗，我也承接了許多自由接案，從設計、前端到後端，逐步發展出更完整的 UI/UX 與全端能力。我相信：良好的架構設計能為企業創造長期價值，遠勝於匆忙上線的短期收益。這也是我這幾年來最大的體悟。

工作之外的我
在生活中，我依然是個熱愛學習、興趣多元的人。從小參與合唱團、美術班與體育活動，到現在依然持續精進，包含歌唱課、體操課，甚至取得了飛輪教練執照。運動之外，我也享受當個「沙發馬鈴薯」，沉浸於追劇時光。

我喜歡讓生活充實多彩，也在學習如何找到生活與工作的平衡。對我來說，這不僅是一個課題，也是能量來源，讓我在專業上持續成長、在生活中保持熱情。
