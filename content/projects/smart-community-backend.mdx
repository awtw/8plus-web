# SmartCommunityBackEnd — 智慧社區後端

![SmartCommunityBackEnd Web](/og/labs/smart-community-backend/web.png)

**Tags**: .NET, PostgreSQL, Redis, Message Queue, Docker

面向智慧社區應用的後端服務，涵蓋住戶、設備、社區服務與事件通報等核心模組，強調可靠性與可觀測性。

## 核心模組
- 使用者/住戶/權限管理
- 設備與感測資料收集
- 通報與工作流程（任務分派、處理、回報）
- 通知中心（Email/SMS/推播）

## 技術棧
- 平台：.NET
- 資料：PostgreSQL、Redis
- 佇列：如 RabbitMQ/Kafka（視部署配置）
- 佈署：Docker/docker-compose
